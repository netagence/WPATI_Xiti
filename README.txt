WPATI_Xiti permet le marquage automatique des pages de WordPress avec le code de suivi AT Internet.

== Description ==
Ce plugin permet d'insérer automatiquement le code de votre compte AT Internet dans chaque page de votre site sous WordPress.

Fonctionnalités actuelles:

*   Paramétrage du compte dans l'administration du site
*   Récupération automatique du nom de la page dans WordPress

Fonctionnalités prévues:

*   Option de marquage des pages par rubrique

== Installation ==

1. Upload  to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does it work with a Xiti account ? =

No, this plugin is made for the professional account from AT Internet, not the free Xiti service.
